Class Validator.GroupBaseField - extends Validator.BaseInputField

This is a group based field and is for allowing inputs to be validated as a group of inputs. An example will be a group of checkboxes where only 1 or 2 should be selected. Or if you want to have a user enter either a username, or an e-mail address.

Configuration Attributes

groupDOM - HTMLElement|string

This is the dom that contains the group's child elements. This is optional as the group's child inputs do not have to be contained in a dom element. Use either the id of the dom object, or the dom object itself.

id - string

Property that can be optional set for the custom input for looking up the object

maxValid - number

Maximum nuymber of fields that can be filled in. Anymore (valid or not) and the group is invalid. If not set, this is property is not used.

membersJSON - Object[]

This will contain the raw json of the input fields that will belong to the group input.

minValid - {number}

Minimum number of inputs that need to be properly filled in for this to be valid. 0 to state that none have to be filled in. If not set, this is property is not used.
