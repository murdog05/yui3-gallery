Configuration Attributes

max - number

This is the maximum allowed value in the double field. Default value is the maximum value for an integer

maxDecimalPlaces - number

If set, this will restrict the number of decimal places allowed on the double. This could be done with regular expression, but this makes it a bit easier for everyone.

maxInclusive - boolean

This is set to true if the maximum allowed values boundary is inclusive

min - number

This is the minimum allowed value in the double field. Default value is the minimum value for an integer

minInclusive - boolean

This is set to true if the minimum allowed values boundary is inclusive
