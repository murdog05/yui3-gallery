Class validator.BaseInputField - extends Base

Known Subclasses:
validator.CustomField validator.TextBaseField validator.CheckboxField validator.GroupBaseField
This class is what all Input fields used in the form validator will inherit from. This provides the basic general attributes and functions required by all input fields.

Configuration Attributes

correctCss - string

If set, this will be applied to the the input to signify that it is correct

correctIndicator - HTMLElement

If set, this will be shown to indicate that the input is correct

correctIndicatorCss - string

This is the css that is to be applied to the indicator. Default will be correctIndicator

correctIndicatorText - string

This will be the text that will be used inside the DOM of the correct indicator. If none is provided, none will be used

createCorrectIndicator - boolean

This will be set to true if the correct indicator is to be created upon instantiation of the input field.

createIncorrectIndicator - boolean

This will be set to true if the Incorrect indicator is to be created upon instantiation of the input field.

disabled - boolean

This is set to true when the input is considered disabled. False otherwise

incorrectCss - string

If set, this will be applied to the the input to signify that it is incorrect

incorrectIndicator - HTMLElement

If set, this will be shown to indicate that the input is incorrect

incorrectIndicatorCss - string

This is the css that is to be applied to the indicator. Default will be incorrectIndicator

incorrectIndicatorText - string

This will be the text that will be used inside the DOM of the incorrect indicator. If none is provided, none will be used

indicatorType - string

This is the DOM element type for the indicator. The default for this will be span.

isOn - boolean

If false, this will signal that the input is considered off, and is not factored into the validation.

optional - boolean

If set, this will show that the input is considered optional, and if not filled in, won't cause the form to be invalid.
