Class Validator.TextBaseField - extends Validator.BaseInputField

Known Subclasses:
Validator.DoubleField Validator.PasswordField Validator.SelectField Validator.MatchField
This class is the text base input field and will be extended by all Inputs who's values will be text based in anyway. Examples include a number input, select box or a text input.

Configuration Attributes

formatter - Function|Object

Formatter used for formatting the result typed into the text field. This can be a function that takes the input text and returns it in a desired format, or it could be an object that has a format function.

inputDOM - HTMLElement|string

This is the main input DOM that the validator will check input on. This can be set as the id of the object, or as the dom object itself.

maxLength - Number

This will state the maximum length of the string in the input. This value is 255 by default.

regex - regex

Regular expression the input must match in order for the input to be correct. If set to null, this is ignored in the isValid function.

textType - string

This property is optional for those who wish to use the pre-canned regular expressions. This can be set to any of the following types
  • Email
  • Phone
  • CreditCard
  • Zipcode
  • Postalcode
