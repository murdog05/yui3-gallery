Form inside a form

Your form doesn't need to be a form to be used with the validator. Nor does it have to be outside another form. As long as it does not have a submit type button in it, it will work fine. Below the first name, last name, age and income are in the main form. The password and password confirm are in the inner form.

For the below example, your better off putting these inputs in the same form. In this case it has been kept simple with familiar inputs. This would be useful, for example, if you have a data table and want an inline form for adding items to it. And that datatable is a part of an overall form.
