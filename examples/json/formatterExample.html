Formatter Example

This will show you how to format the fields in your form. In this example we have a currency field and a phone number field that we want formatted.

For the income a custom formatter was used, while the a built in one was used for the phone number. The regular expression will be used on both to determine if the input is valid. If it is the formatter will then take in the valid text entered into the text field and format it properly. The income should format to look like dollars, while the phone number should be of this format (xxx) xxx-xxxx
